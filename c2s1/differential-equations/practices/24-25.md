### Заняття 24&ndash;25: Варіаційне числення

#### Рекомендовані приклади для аудиторної роботи

Знайти екстремалі таких функціоналів:

1. $$I[y(x)] = \Int_{-1}^0 (12 x y - (y')^2) \diff x$$, $$y(-1) = 1$$, $$y(0) = 0$$.

2. $$I[y(x)] = \Int_{-1}^1 ((y')^2 - 2 x y) \diff x$$, $$y(-1) = -1$$, $$y(1) = 1$$.

3. $$I[y(x)] = \Int_0^1 (y^2 + 2 (y')^2 + (y'')^2) \diff x$$, $$y(0) = 0$$, $$y(1) = 0$$, $$y'(0) = 1$$, $$y'(1) = - \sinh (1)$$.

4. $$I[y(x)] = \Int_0^1 (y + y'') \diff x$$, $$y(0) = y_0$$, $$y(1) = y_1$$, $$y'(0) = y_0'$$, $$y'(1) = y_1'$$.

5. $$I[y(x)] = \Int_0^1 ((y')^2 + (y'')^2) \diff x$$, $$y(0) = 0$$, $$y(1) = \sinh (1)$$, $$y'(0) = 1$$, $$y'(1) = \cosh (1)$$.

6. $$I[y(x), z(x)] = \Int_0^{\pi / 4} (2 z - 4 y^2 + (y')^2 - (z')^2) \diff x$$, $$y(0) = 0$$, $$y(\pi / 4) = 1$$, $$z(0) = 0$$, $$z(\pi / 4) = 1$$.

7. $$I[y(x), z(x)] = \Int_0^{\pi / 2} ((y')^2 + (z')^2 - 2 y z) \diff x$$, $$y(0) = 0$$, $$y(\pi / 2) = 1$$, $$z(0) = 0$$, $$z(\pi / 2) = 1$$.

	Дослідити на екстремум функціонали:

8. $$I[y(x)] = \Int_0^1 ((y')^3 + y') \diff x$$, $$y(0) = 0$$, $$y(1) = 2$$.

9. $$I[y(x), z(x)] = \Int_0^1 ((y')^2 + (z')^2) \diff x$$, $$y(0) = 0$$, $$y(1) = 1$$, $$z(0) = 0$$, $$z(1) = 2$$.

#### Рекомендовані приклади для домашнього завдання

Знайти екстремалі таких функціоналів:

1. $$I[y(x)] = \Int_0^1 \sqrt{y \cdot (1 + (y')^2)} \diff x$$, $$y(0) = y(1) = 1 / \sqrt{2}$$.

2. $$I[y(x)] = \Int_{-1}^1 ((y')^2 - 2 x y) \diff x$$, $$y(-1) = -1$$, $$y(1) = 1$$.

3. $$I[y(x)] = \Int_{-1}^0 (240 y - (y''')^2) \diff x$$, $$y(-1) = 1$$, $$y(0) = 0$$, $$y'(-1) = -4.5$$, $$y'(0) = 0$$, $$y''(-1) = 16$$, $$y''(0) = 0$$.

4. $$I[y(x)] = \Int_a^b ((y')^2 + y \cdot y'') \diff x$$, $$y(a) = A_1$$, $$y(b) = B_1$$, $$y'(a) = A_2$$, $$y'(b) = B_2$$.

5. $$I[y(x)] = \frac{1}{2} \Int_0^1 (y'')^2 \diff x$$, $$y(0) = 0$$, $$y(1) = 1$$, $$y'(0) = 0$$, $$y'(1) = 1$$.

6. $$I[y(x), z(x)] = \Int_{-1}^1 ( 2 x y - (y')^2 + (z')^3 / 3 ) \diff x$$, $$y(1) = 0$$, $$y(-1) = 2$$, $$z(1) = 1$$, $$z(-1) = -1$$.

7. $$I[y(x), z(x)] = \Int_0^1 ((y')^2 + (z')^2 + 2 y) \diff x$$, $$y(0) = 1$$, $$y(1) = 3 / 2$$, $$z(0) = 0$$, $$z(1) = 1$$.

	Дослідити на екстремум функціонали:

8. $$I[y(x)] = \Int_0^1 (y + 2 x + (y')^2 / 2) \diff x$$, $$y(0) = y'(0) = 0$$.

9. $$I[y(x), z(x)] = \Int_0^1 ((y')^2 + (z')^2 + 4 z) \diff x$$, $$y(0) = 0$$, $$y(1) = 1$$, $$z(0) = 0$$, $$z(1) = 0$$.
